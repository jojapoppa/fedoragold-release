All fedoragold_daemon calls are JSON and are shown below:

Example GET (daemon height): getheight
curl -X GET -i -H "Accept: application/json" -d '{"jsonrpc": "2.0"}' http://localhost:30159/getheight

getheight
  function: get the height
    response.height
    response.status

iscoreready
  function: is the daemon ready for proessing
    response.Iscoreready
    response.status

getblocks.bin
  function: quickly get the blocks by id
    request.blockids
    response.blocks
    response.start_height
    response.current_height
    response.status

    request.txid
    response.o_indexes
    response.status
